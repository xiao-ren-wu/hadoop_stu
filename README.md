# hadoop
-----------------------------------------
**hadoop中有3个核心组件**
- 分布式文件系统：HDFS —— 实现将文件分布式存储在很多的服务器上
- 分布式运算编程框架：MAPREDUCE —— 实现在很多机器上分布式并行运算
- 分布式资源调度平台：YARN —— 帮用户调度大量的mapreduce程序，并合理分配运算资源
